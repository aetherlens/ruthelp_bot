# ruthelp_bot 

[Ссылка на бота](https://t.me/ruthelp_bot)

## Installation  

```
sudo apt update && sudo apt install -y python3.8 python3-venv
git clone https://github.com/ekiauhce/ruthelp_bot
cd ruthelp_bot/
. setup.sh
```

## Before launch

Change `TG_API_TOKEN` and `HOST_IP` in `run.sh` file.

## How to run

`. run.sh`
